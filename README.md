# TutoringEmailAutomation
Automated agents for generating, formatting, and sending personalized tutoring emails for Thara_courses.
This repository contains a set of AI-powered agents designed to automate the process of communicating with clients for private tutoring services.

The system can:
-Generate professional and polite cold emails offering tutoring sessions.
-Personalize emails with student names, subjects, and available time slots.
-Convert text emails into clean HTML for better readability.
-Automatically send emails via SendGrid.
-Ensure message safety by filtering inappropriate language or content.

Built using OpenAI agents, Pydantic for structured outputs, and SendGrid for email delivery. Ideal for educational services looking to automate client communications while maintaining a professional touch.
